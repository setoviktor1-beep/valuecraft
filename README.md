# ValueCraft SaaS Template

A premium static multi-page website designed for SaaS products focusing on value propositions.

## Structure
```text
valuecraft/
├── index.html          # Home (Hero & Problem)
├── features.html       # Features (Outcome-focused)
├── pricing.html        # Pricing (3 Tiers)
├── use-cases.html      # Use Cases (Scenarios)
├── faq-contact.html    # FAQ & Contact (Mailto)
├── assets/             # Images & Logos
├── css/
│   └── custom.css      # Animations & Glass styles
└── js/
    └── main.js         # Interactivity
```

## Features
- **Modern Dark Theme**: Slate-based palette with Indigo accents.
- **Animated Social Icons**: Custom inline SVG with float & hover animations.
- **Responsive Navigation**: Sticky header with mobile menu toggle.
- **FAQ Accordion**: Custom vanilla JS toggle.
- **Contact Form**: Direct `mailto` integration.
